canvas playground

a little introduction to the canvas playground

gravity

If bool is set to 1 all points will start falling down. You can only play with the strength, the tick function is not active any more. By set it to false again all points will start to float again.

start

The initial position for each point where i is the index of a point. Any change will only be noticed if you hit the reset button.

tick(1/60sek best)

is called every frame for each point where i is the index of a point.

speed

speed for each point where i is the index of a point.
